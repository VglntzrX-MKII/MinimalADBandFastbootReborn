**ℹ️ ABOUT THIS TOOL**

This is a redistribution of the official Android SDK Platform Tools.

**ADB (Android Debug Bridge):** A versatile command-line tool that lets
you communicate with a device.

**Fastboot:** A diagnostic tool used to modify the Android filesystem
from a computer when the device is in bootloader mode.

Both tools are part of **Google's Android Open-Source Project (AOSP)**.
